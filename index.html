Pertanyaan Populer

Hal yang sering ditanyakan calon klien sebelum memulai proyek.

Berapa lama waktu pembuatan website?

Tergantung pada kompleksitas proyek. Untuk company profile standar biasanya memakan waktu 1-2 minggu. Sedangkan untuk e-commerce atau custom web app bisa memakan waktu 3-6 minggu.

Apakah ada biaya bulanan/tahunan?

Ya, untuk menjaga website Anda tetap online, terdapat biaya tahunan untuk perpanjangan Domain dan Hosting. Biayanya sangat terjangkau dan akan kami informasikan transparan di awal.

Apakah saya bisa mengubah konten web sendiri nanti?

Tentu saja. Kami akan menyediakan halaman Admin Dashboard (CMS) yang mudah digunakan. Anda juga akan mendapatkan panduan/tutorial cara mengelola konten web Anda.

Apakah melayani pembuatan desain logo juga?

Ya, kami memiliki tim desainer grafis. Pembuatan logo dapat ditambahkan sebagai layanan ekstra di luar paket pembuatan website utama.
